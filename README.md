# PeopleDetectionTest

Тестовое задание по созданию программы для детекции людей на видео.
Программа использует [**YOLOv8**](https://huggingface.co/Ultralytics/YOLOv8), PyQT6 для интерфейса и OpenCV для работы с видео.

---

## Установка

```bash
git clone https://github.com/saigedoc/PeopleDetectionTest.git
cd PeopleDetectionTest
pip install -r requirements.txt
```

**Примечание:** Для достаточно быстрой обработки, при наличие GPU, следует установить [CUDA Toolkit](https://developer.nvidia.com/cuda-toolkit), а также специальную версию [Pytorch](https://pytorch.org/get-started/locally/), поддержвиющую конкретную CUDA.

## Инструкция по использованию

При запуске откроется окно интерфейса.
Верхние три кнопки отвечают за выбор путей к файлам (весов, видео на вход, путь сохранения обработанного видео).
После выбора всех трёх путей следует нажать кнопку "Process video" для старта обработки видео.
Какое-то время видео будет обрабатываться, можно наблюдать процент готовности на прогресс баре, по звершению откроется окно уведомления о завешении обработки.
